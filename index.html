patchlib

A library for working with N-D patches, currently very much in development. Please do a recursive pull, as the git has git submodules mgt and mvit.

Please let us know of any questions/suggestions at adalca@csail.mit.edu

Main Functions

  • patchlib.vol2lib transform a N-D volume into a patch library
  • patchlib.volStruct2lib transform a volStruct into a patch library
  • patchlib.grid grid of patch starting points for N-D volume
  • patchlib.stackPatches stack patches in layer structure
  • patchlib.volknnsearch k-NN search of patches in source given a set of reference volumes
  • patchlib.patchmrf mrf patch inference on patch candidates on a grid
  • patchlib.quilt quilt or reconstruct volume from patch indexes in library
  • patchlib.gridsize grid of patches that fit into a volume

Helpers

  • patchlib.overlapkind overlap amount to/from pre-specified overlap kind
  • patchlib.guessPatchSize guess the size of a patch from nVoxels
  • patchlib.patchCenterDist compute the distance to the center of a patch
  • patchlib.gridsize2volSize volume size from number of patches
  • patchlib.isvalidoverlap check overlap variable
  • patchlib.grid2volSize olume size from patch grid size
  • patchlib.patchesmat2cell
  • patchlib.l2overlapdst
  • patchlib.lib2patches (still in draft mode)

Examples

  • example_quilt test quilting/reconstruction
  • example_viewPatchesInImage test view.patchesInImage
  • example_viewPatchMatches2D test view.patchMatches2D
  • example_viewPatchRef2D test view.patchRef2D

  • example_stackPatches Test stackPatches on simple reconstruction task.
  • example_patchmrf Test patchmrf

Visualization

  • patchview.patchesInImage visualize 2D patches in an image
  • patchview.patchMatches2D display 2D patches matching an original patch
  • patchview.patches2D show 2D patches in a subplot grid
  • patchview.layers2D view layers (as returned by patchlib.stackPatches)
  • patchview.patchRef2D
  • patchview.drawPatchRect draw rectangles in the current axis
